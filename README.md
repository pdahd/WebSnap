### 📖 项目简介
WebSnap 允许你在安卓设备上轻松截取完整的网页长图。

### 🚀 快速开始
1. 在 **Releases** 页面下载最新的 APK。
2. 安装并输入网址。
3. 点击“截图”即可保存长图。
